Definitions

High-level A high-level programming language offers a strong abstraction from the details of the computer.

Î™nterpreted An interpreted programming language is a language such that programs written in it, don't need to be compiled to a machine language program in order to be executed.

General-purpose A general-purpose programming language is designed to be used for writing software in a wide variety of application domains.

Programming paradigm A programming paradigm is a general way to write code. Programming paradigms are used to classify programming languages.

Structured programming Structured programming is a programming paradigm that makes wide use of control flow constructs (selection and repetition), block structures and subroutines, aiming to improve code quality and development time.

Functional programming Functional programming is the programming paradigm in which programs are mostly constructed by applying and composing functions.

Object-oriented programming Object-oriented programming is a programming paradigm that focuses on using "objects", which is an entity that encapsulates data and procedures.

Low-level Low-level when reffering to programming, means something closely related to (computer specific) hardware.

Unicode A standard for encoding text. It covers the characters of most existing human languages and many special symbols.

Garbage collection Garbage collection in Computer Science is a way of managing memory. Programming languages that use a grabage collector save the programmer from having to take care of memory management by themselves.

Whitespace indentation This kind of indentation means that there exists an increase in indentation after certain statements and a decrease in indentation when a block ends.

Scripting language A scripting language is a programming language that is used (among other things) to automate tasks that would normally be executed one by one by a human. Usually a scripting language is also an interpreted one.

Scientific computing Scientific computing (also called Computational Science) is a field that uses advanced computing techniques to solve complex problems.

Machine learning Machine learning is a subfield of artificial intelligence that focuses on creating algorithms that improve automatically through experience and through the use of data.

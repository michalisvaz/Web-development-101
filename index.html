Τεχνολογίες και Προγραμματισμός Εφαρμογών στον Ιστό

Οι σημαντικότερες γνώσεις που αποκόμισα από την εργασία ήταν η χρήση CSS για τη δημιουργία του Layout μιας ιστοσελίδας και για το Responsiveness αυτής, καθώς και η χρήση JavaScript για το 'πείραγμα' του περιεχομένου και της μορφοποίησής του. Οι σημαντικότερες δυσκολίες που αντιμετώπισα είχαν να κάνουν με την μορφοποίηση και την διάταξη του περιεχομένου της ιστοσελίδας. Κατά τη διάρκεια αυτών των διαδικασιών πολύ συχνά αφιέρωνα πολύ χρόνο για λεπτομέρειες που ως απλός χρήστης θα μου φαίνονταν ασήμαντες (αλλά ως προγραμματιστής/σχεδιαστής πρέπει να λάβει κανείς υπόψη του, γιατί μπορεί να είναι σημαντικές για κάποιους χρήστες).
